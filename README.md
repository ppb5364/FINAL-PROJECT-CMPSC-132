# FINAL-PROJECT-CMPSC-132

## Project Description 
The project I have done is a two-player Tic-Tac-Toe game written in Python. It starts of by creating a 3x3 board and allows players X and O to 
take turns choosing the row no. and column no. to place their move. After each turn, the board updates and the game checks for a win through rows, 
columns, or diagonals. It also detects a draw when the board is full.
